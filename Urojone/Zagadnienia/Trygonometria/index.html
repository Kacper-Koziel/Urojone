Postać trygonometryczna

Poza postacią algebraiczną, liczby zespolone mają również postać trygonometryczną przedstawioną przez r(cos φ + i sin φ) gdzie r to moduł liczby zespolonej a argument φ to kąt między osią rzeczywistą a promieniem wodzącym liczby zespolonej na płaszczyźnie zespolonej.

Tangens kąta φ

W celu wyznaczenia argumentu φ, najpierw musimy policzyć jego tangens by potem móc modyfikować go przez poszczegulne przypadki. Wzór na tangens dla z = a + bi, to b / a. Dla z = a - bi, tan φ = -b / a.

Arcus Tangens kąta φ

Kiedy posiadamy tan φ, używamy funkcji odwrotnej do tangensa czyli arcus tangensa. Niestety liczenie go graniczy z cudem, dlatego też współcześnie używa się kalkulatorów. Jeśli jednak bardzo nie chcemy sobie ułatwiać życia zawsze jest szereg Taylora który pozwala nam z kilku pierwszych składników policzyć przybliżony arctang(φ).

Modyfikacje kąta φ

Niestety arctan ma przedział od -90° do 90°, co oznacza że nie zawsze daje poprawny wynik. Dlatego też czasem musimy go modyfikować według zasad:
dla a > 0 => nie modyfikujemy,
dla a < 0 => dodajemy 180°

Specjalne przypadki

Dla a = 0:
b > 0 => φ = 90°,
b < 0 => φ = -90°

Dla b = 0:
a > 0 => φ = 0°,
a < 0 => φ = 180°

Dla a = 0 i b = 0 => niezdefiniowane

Po modyfikacjach kąta mamy już nasz argument φ. Wystarczy wstawić go do wzoru

Zastosowanie postaci trygonometrycznej

Poza wizualnym przedstawieniem liczb na płaszczyźnie zespolonej, postać trygonometryczna niezwykle ułatwia działania takie jak: mnożenie, dzielenie oraz potęgowanie.

Mnożenie

Dzięki postaci trygonometrycznej mnożenie możemy wykonać poprzez pomnożenie modułów oraz wykonanie sinusa i cosinusa z sumy argumentów. Pozwala nam to na znaczne przyspieszenie tego dość długotrwałego działania. Kiedy suma kątów wyjdzie poza limit -180° do 180°, musimy dodać lub odjąć 360°, aby wynik mieścił się w tym przedziale.

Dzielenie

Dzielenie które było niezwykle żmudnym procesem, dzięki postaci trygonometrycznej stało się szybkim uzupełnieniem wzoru. Wystarczy podzielić dwa moduły i wykonać sinusa i cosinusa z różnicy argumentów. Kiedy różnica kątów wyjdzie poza limit -180° do 180°, musimy dodać lub odjąć 360°, aby wynik mieścił się w tym przedziale.

Potęgowanie

Postać trygonometryczna pozwala nam na niezwykle proste potęgowanie do dowolnego stopnia, bez używania wzorów skróconego mnożenia. Po podniesieniu modułu do potęgi n (stopnia do którego podnosimy) wystarczy wykonać wykonać sinusa i cosinusa z wyniku mnożenia n i argumentu.
